    Ch.4 мң мӮ¬ мӮ¬лЎҖмҷҖ нӮӨмӣҢл“ң м •лҰ¬¶

    < Stack Frameкіј Virtual Memory

    мқҙлІҲ мұ•н„°м—җм„ңлҠ” н”„лЎңм„ёмҠӨмҷҖ мҠӨл Ҳл“ңк°Җ л©”лӘЁлҰ¬м—җм„ң м–ҙл–»кІҢ мЎҙмһ¬н•ҳлҠ”м§Җ нҷ•мқён–ҲлӢӨ. ProcessPool 16к°ңк°Җ 1GBлҘј лЁ№лҠ” мқҙмң , мһ¬к·Җ 1000лІҲмқҙ мЈҪлҠ” мқҙмң лҘј Memory Layout, Virtual Memory, Page FaultлЎң м„ӨлӘ…н–ҲлӢӨ.

    к°ҷмқҖ мӣҗлҰ¬к°Җ м Ғмҡ©лҗҳлҠ” мӢӨл¬ҙ мӮ¬лЎҖлҘј лӘҮ к°Җм§Җ лҚ” ліёлӢӨ.

    4-6. мң мӮ¬ мӮ¬лЎҖ¶

    Docker м»Ён…Ңмқҙл„Ҳ л©”лӘЁлҰ¬ м ңн•ң¶

    Docker(лҳҗлҠ” K8s)м—җм„ң м»Ён…Ңмқҙл„Ҳм—җ л©”лӘЁлҰ¬ м ңн•ңмқ„ кұҙлӢӨ:

    docker run --memory=512m my-app

    м»Ён…Ңмқҙл„Ҳ м•Ҳмқҳ н”„лЎңм„ёмҠӨк°Җ 512MBлҘј мҙҲкіјн•ҳл©ҙ? OOM Killerк°Җ м»Ён…Ңмқҙл„ҲлҘј мЈҪмқёлӢӨ.

    K8sм—җм„ңлҠ” resources.limits.memoryлЎң лҸҷмқјн•ң м ңн•ңмқ„ кұҙлӢӨ:

    resources:
  limits:
    memory: "512Mi"

    "мҷң Podмқҙ мһҗкҫё Restart лҗҳлҠ”к°Җ"мқҳ нқ”н•ң мӣҗмқёмқҙ л©”лӘЁлҰ¬ м ңн•ң мҙҲкіјлӢӨ. ProcessPool мӣҢм»ӨлҘј л§Һмқҙ лқ„мҡ°л©ҙм„ң м»Ён…Ңмқҙл„Ҳ л©”лӘЁлҰ¬ м ңн•ңмқ„ кі л Өн•ҳм§Җ м•Ҡмңјл©ҙ, л°°нҸ¬н•ҳмһҗл§Ҳмһҗ OOMмңјлЎң мЈҪлҠ” мғҒнҷ©мқҙ лІҢм–ҙм§„лӢӨ.

    Javaмқҳ -XmxмҷҖ Pythonмқҳ м°Ёмқҙ¶

    JavaлҠ” JVM мӢңмһ‘ мӢң Heap нҒ¬кё°лҘј лӘ…мӢңм ҒмңјлЎң м„Өм •н•ңлӢӨ:

    java -Xms256m -Xmx512m MyApp

    (Python мӮ¬мҡ©мһҗлҘј мң„н•ҙ: -XmsлҠ” мҙҲкё° Heap нҒ¬кё°, -XmxлҠ” мөңлҢҖ Heap нҒ¬кё°лӢӨ.)

    Heapмқҙ -XmxлҘј мҙҲкіјн•ҳл©ҙ OutOfMemoryErrorк°Җ мҰүмӢң л°ңмғқн•ңлӢӨ. кІҪкі„к°Җ лӘ…нҷ•н•ҳлӢӨ.

    PythonмқҖ? мқҙлҹ° лӘ…мӢңм Ғ м ңн•ңмқҙ м—ҶлӢӨ. Heapмқҙ мӢңмҠӨн…ңмқҙ н—Ҳмҡ©н•ҳлҠ” н•ң кі„мҶҚ мһҗлһҖлӢӨ. л©”лӘЁлҰ¬ м ңн•ңмқ„ кұёл Өл©ҙ OS л ҲлІЁ(resource.setrlimit())мқҙлӮҳ м»Ён…Ңмқҙл„Ҳ л ҲлІЁ(--memory)лЎң н•ҙм•ј н•ңлӢӨ. Pythonмқҳ OOMмқҙ лҚ” мҳҲмёЎн•ҳкё° м–ҙл Өмҡҙ мқҙмң к°Җ мқҙкұ°лӢӨ.

    Gunicorn мӣҢм»Ө мҲҳ кІ°м •¶

    Python мӣ№ м„ңлІ„лЎң Gunicornмқ„ м“ё л•Ң, мӣҢм»Ө мҲҳлҘј м„Өм •н•ңлӢӨ:

    gunicorn -w 4 myapp:app
    Gunicorn Python WSGI м„ңлІ„лӢӨ. н”„лЎңм„ёмҠӨ кё°л°ҳмңјлЎң лҸҷмһ‘н•ңлӢӨ. `-w 4`лҠ” мӣҢм»Ө н”„лЎңм„ёмҠӨлҘј 4к°ң лқ„мҡҙлӢӨлҠ” лң»мқҙлӢӨ. к°Ғ мӣҢм»Өк°Җ лҸ…лҰҪ н”„лЎңм„ёмҠӨмқҙлҜҖлЎң, мқҙлІҲ мұ•н„°м—җм„ң ліё кІғмІҳлҹј мӣҢм»Ө мҲҳ x Python лҹ°нғҖмһ„ нҒ¬кё°л§ҢнҒј л©”лӘЁлҰ¬лҘј лЁ№лҠ”лӢӨ.

    Gunicorn кіөмӢқ л¬ём„ңк°Җ к¶ҢмһҘн•ҳлҠ” мӣҢм»Ө мҲҳ кіөмӢқ: (2 x CPU мҪ”м–ҙ мҲҳ) + 1. 4мҪ”м–ҙ м„ңлІ„лқјл©ҙ 9к°ң.

    мҷң мқҙ кіөмӢқмқёк°Җ? CPU мҪ”м–ҙ мҲҳліҙлӢӨ мӣҢм»Өк°Җ л§Һмңјл©ҙ мӢӨм ң лі‘л ¬ мӢӨн–ү мқҙм җ м—Ҷмқҙ л©”лӘЁлҰ¬л§Ң лӮӯл№„н•ңлӢӨ. мҪ”м–ҙ мҲҳмқҳ 2л°° + 1 м •лҸ„к°Җ CPU нҷңмҡ©кіј л©”лӘЁлҰ¬ л№„мҡ©мқҳ к· нҳ•м җмқҙлқјлҠ” кІҪн—ҳм Ғ кіөмӢқмқҙлӢӨ.

    (л¬ҙн„ұлҢҖкі  -w 32лЎң мҳ¬лҰ¬лҠ” мӮ¬лһҢмқҙ к°ҖлҒ” мһҲлӢӨ. мқҙлІҲ мұ•н„°лҘј мқҪм—ҲлӢӨл©ҙ мҷң к·ёкІҢ мң„н—ҳн•ңм§Җ мқҙн•ҙн•  кұ°лӢӨ.)

    fork()мҷҖ Copy-on-Write¶

    ProcessPoolмқҙ мӣҢм»ӨлҘј л§Ңл“Ө л•Ң лӮҙл¶Җм ҒмңјлЎң fork() System Callмқ„ м“ҙлӢӨ.

    fork() Unix/Linux/macOSм—җм„ң н”„лЎңм„ёмҠӨлҘј ліөм ңн•ҳлҠ” System CallмқҙлӢӨ. л¶ҖлӘЁ н”„лЎңм„ёмҠӨмқҳ л©”лӘЁлҰ¬ кіөк°„мқ„ мһҗмӢқ н”„лЎңм„ёмҠӨм—җкІҢ ліөмӮ¬н•ңлӢӨ. лӢЁ, нҳ„лҢҖ OSлҠ” Copy-on-Write(CoW)лҘј м“ҙлӢӨ. fork() м§Ғнӣ„м—җлҠ” мӢӨм ңлЎң л©”лӘЁлҰ¬лҘј ліөмӮ¬н•ҳм§Җ м•Ҡкі , л¶ҖлӘЁмҷҖ мһҗмӢқмқҙ к°ҷмқҖ л¬јлҰ¬ л©”лӘЁлҰ¬ PageлҘј кіөмң н•ңлӢӨ. м–ҙлҠҗ н•ңмӘҪмқҙ PageлҘј мҲҳм •н•  л•Ң л№„лЎңмҶҢ н•ҙлӢ№ Pageл§Ң ліөмӮ¬н•ңлӢӨ. к·ёлһҳм„ң fork() м§Ғнӣ„м—җлҠ” л©”лӘЁлҰ¬к°Җ кұ°мқҳ мҰқк°Җн•ҳм§Җ м•Ҡм§Җл§Ң, мӣҢм»Өк°Җ мӢӨм ңлЎң мһ‘м—…мқ„ мӢңмһ‘н•ҳл©ҙ(лҚ°мқҙн„°лҘј м“°л©ҙ) л©”лӘЁлҰ¬к°Җ кёүмҰқн•ңлӢӨ. м°ёкі : macOSмқҳ Python 3.8 мқҙнӣ„ `multiprocessing`мқҳ кё°ліё start methodлҠ” `fork`к°Җ м•„лӢҢ `spawn`мқҙлӢӨ. `spawn`мқҖ fork() лҢҖмӢ  мғҲ Python мқён„°н”„лҰ¬н„° н”„лЎңм„ёмҠӨлҘј мІҳмқҢл¶Җн„° мӢңмһ‘н•ҳлҜҖлЎң, л¶ҖлӘЁ н”„лЎңм„ёмҠӨмқҳ мғҒнғңлҘј ліөм ңн•ҳм§Җ м•ҠлҠ”лӢӨ. мқҙлІҲ мұ•н„°мқҳ мӢӨмҠө нҷҳкІҪ(M1 Mac, Python 3.12)м—җм„ң ProcessPoolExecutorлҠ” `spawn`мңјлЎң лҸҷмһ‘н•ңлӢӨ.

    м•һм—җм„ң ProcessPool мӣҢм»Өмқҳ RSSк°Җ к°Ғк°Ғ ~60MBмҳҖлҠ”лҚ°, мқҙкұҙ мӣҢм»Өк°Җ Python мқён„°н”„лҰ¬н„°лҘј мӢӨн–үн•ҳл©ҙм„ң лҚ°мқҙн„°лҘј м“°кё° мӢңмһ‘н•ң мқҙнӣ„мқҳ к°’мқҙлӢӨ. fork() м§Ғнӣ„м—җлҠ” Copy-on-Write лҚ•л¶„м—җ л¬јлҰ¬ л©”лӘЁлҰ¬лҘј лҚң лЁ№м§Җл§Ң, мӢӨм§Ҳм Ғмқё мһ‘м—…мқҙ мӢңмһ‘лҗҳл©ҙ Pageк°Җ ліөмӮ¬лҗҳл©ҙм„ң л©”лӘЁлҰ¬к°Җ лҠҳм–ҙлӮңлӢӨ.

    к·ёлһҳм„ң мӢӨл¬ҙм—җм„ңлҠ” м–ҙл–»кІҢ н•ҳлҠ”к°Җ¶

    1. н”„лЎңм„ёмҠӨ мӣҢм»Ө мҲҳлҠ” CPU мҪ”м–ҙ мҲҳ кё°л°ҳмңјлЎң кІ°м •н•ңлӢӨ¶

    import os

cpu_count = os.cpu_count()

# CPU Bound мһ‘м—…: мҪ”м–ҙ мҲҳ мқҙн•ҳ
process_workers = cpu_count

# I/O Bound мһ‘м—…: мҪ”м–ҙ мҲҳ x 2 м •лҸ„
io_workers = cpu_count * 2

    Ch.3м—җм„ң "CPU Boundм—җлҠ” ProcessPool"мқҙлқјкі  н–Ҳм§Җл§Ң, мӣҢм»Ө мҲҳлҠ” мҪ”м–ҙ мҲҳлҘј л„ҳкё°м§Җ м•ҠлҠ” кІҢ м•Ҳм „н•ҳлӢӨ. мҪ”м–ҙ мҲҳлҘј мҙҲкіјн•ҳл©ҙ мӢӨм ң лі‘л ¬ мІҳлҰ¬ мқҙм җ м—Ҷмқҙ л©”лӘЁлҰ¬л§Ң лӮӯл№„н•ңлӢӨ.

    2. мһҗлЈҢкө¬мЎ°лҘј нҷңмҡ©н•ҙм„ң мһ¬к·ҖлҘј м—Ҷм•ӨлӢӨ¶

    мӮ¬лЎҖ Bм—җм„ң лҙӨл“Ҝмқҙ, dictм—җ ID кё°л°ҳмңјлЎң м ҖмһҘлҗң лҚ°мқҙн„°лҘј көімқҙ лЈЁнҠёл¶Җн„° мһ¬к·ҖлЎң нғҗмғүн•  мқҙмң к°Җ м—ҶлӢӨ:

    # лЈЁнҠёл¶Җн„° мһ¬к·Җ DFS (Stack Overflow мң„н—ҳ, O(N x D))
def _find_from_root_recursive(tree, target_id, current_id=0):
    if current_id == target_id:
        return [tree[current_id]["name"]]
    children = [cid for cid, cat in tree.items() if cat["parent_id"] == current_id]
    for child_id in children:
        result = _find_from_root_recursive(tree, target_id, child_id)
        if result is not None:
            return [tree[current_id]["name"]] + result
    return None

# IDлЎң м§Ғм ‘ м ‘к·ј + parent л”°лқј мҳ¬лқјк°Җкё° (Stack Overflow м—ҶмқҢ, O(depth))
def _find_by_direct_lookup(tree, target_id):
    path = []
    current = target_id
    while current is not None:
        path.append(tree[current]["name"])
        current = tree[current]["parent_id"]
    return list(reversed(path))

    нҠёлҰ¬ нғҗмғүмІҳлҹј мһ¬к·Җк°Җ мһҗм—°мҠӨлҹ¬мҡҙ кІҪмҡ°м—җлҸ„, лӘ…мӢңм Ғ Stack(лҰ¬мҠӨнҠё)мқ„ мӮ¬мҡ©н•ҳл©ҙ Heapм—җ лҚ°мқҙн„°лҘј мҢ“мңјлҜҖлЎң Stack нҒ¬кё° м ңн•ңм—җ кұёлҰ¬м§Җ м•ҠлҠ”лӢӨ:

    # мһ¬к·Җ DFS (Stackм—җ Frameмқҙ мҢ“мһ„)
def dfs_recursive(node):
    visit(node)
    for child in node.children:
        dfs_recursive(child)

# л°ҳліө + лӘ…мӢңм Ғ Stack (Heapм—җ мҢ“мһ„ вҶ’ Stack Overflow м—ҶмқҢ)
def dfs_iterative(root):
    stack = [root]
    while stack:
        node = stack.pop()
        visit(node)
        stack.extend(node.children)

    3. л©”лӘЁлҰ¬ мӮ¬мҡ©лҹүмқ„ лӘЁлӢҲн„°л§Ғн•ңлӢӨ¶

    import resource
import sys

def get_rss_mb():
    usage = resource.getrusage(resource.RUSAGE_SELF)
    if sys.platform == "darwin":
        return usage.ru_maxrss / (1024 * 1024)  # macOS: bytes
    return usage.ru_maxrss / 1024  # Linux: KB

    м„ңлІ„ мҡҙмҳҒм—җм„ңлҠ” Prometheus + Grafana к°ҷмқҖ лҸ„кө¬лЎң н”„лЎңм„ёмҠӨ л©”лӘЁлҰ¬лҘј мғҒмӢң лӘЁлӢҲн„°л§Ғн•ңлӢӨ. RSSк°Җ кҫёмӨҖнһҲ мҰқк°Җн•ҳлҠ” нҢЁн„ҙмқҙ ліҙмқҙл©ҙ л©”лӘЁлҰ¬ лҲ„мҲҳлҘј мқҳмӢ¬н•ҙм•ј н•ңлӢӨ.

    4. GeneratorлЎң Heapмқ„ м•„лӮҖлӢӨ¶

    лҢҖмҡ©лҹү лҚ°мқҙн„°лҘј мІҳлҰ¬н•  л•Ң, м „л¶Җ л©”лӘЁлҰ¬м—җ мҳ¬лҰ¬лҠ” лҢҖмӢ  GeneratorлҘј м“°л©ҙ Heap мӮ¬мҡ©лҹүмқ„ к·№м ҒмңјлЎң мӨ„мқј мҲҳ мһҲлӢӨ:

    # лҰ¬мҠӨнҠё: 100л§Ң к°ңлҘј н•ңлІҲм—җ Heapм—җ мҳ¬лҰј (~39 MB)
data = [process(item) for item in range(1_000_000)]

# Generator: н•ҳлӮҳм”© мІҳлҰ¬, Heapм—җ н•ң к°ңл§Ң мң м§Җ
data = (process(item) for item in range(1_000_000))

    (м•һм—җм„ң мёЎм •н•ң Heap мҰқк°Җ нҢЁн„ҙмқ„ л– мҳ¬л Өліҙмһҗ. 100л§Ң к°ң лҰ¬мҠӨнҠё = ~39MBмҳҖлӢӨ. GeneratorлҘј м“°л©ҙ мқҙкұё кұ°мқҳ 0м—җ к°Җк№қкІҢ мӨ„мқј мҲҳ мһҲлӢӨ.)

    3. мҳӨлҠҳ нӮӨмӣҢл“ң м •лҰ¬¶

    CSлҘј нӮӨмӣҢл“ңлЎң л°°мҡҙлӢӨлҠ” кұҙ, к°ңлі„ к°ңл…җмқ„ мҷёмҡ°лҠ” кІҢ м•„лӢҲлқј к°ңл…җл“Ө мӮ¬мқҙмқҳ м—°кІ°мқ„ мқҙн•ҙн•ҳлҠ” кұ°лӢӨ. мқҙлІҲ мұ•н„°м—җм„ң лӮҳмҳЁ нӮӨмӣҢл“ңл“Өмқ„ лӘЁм•„м„ң м •лҰ¬н•ңлӢӨ.

    Process (н”„лЎңм„ёмҠӨ) мӢӨн–ү мӨ‘мқё н”„лЎңк·ёлһЁмқҳ мқёмҠӨн„ҙмҠӨлӢӨ. мҡҙмҳҒмІҙм ңлЎңл¶Җн„° лҸ…лҰҪм Ғмқё л©”лӘЁлҰ¬ кіөк°„(Text, Data, Heap, Stack)кіј мӢңмҠӨн…ң мһҗмӣҗ(File Descriptor л“ұ)мқ„ н• лӢ№л°ӣлҠ”лӢӨ. PCBлЎң кҙҖлҰ¬лҗңлӢӨ. н”„лЎңм„ёмҠӨ к°„м—җлҠ” л©”лӘЁлҰ¬к°Җ мҷ„м „нһҲ кІ©лҰ¬лҗҳм–ҙ мһҲм–ҙм„ң м•Ҳм „н•ҳм§Җл§Ң, к·ёл§ҢнҒј мғқм„ұ л№„мҡ©кіј л©”лӘЁлҰ¬ л№„мҡ©мқҙ нҒ¬лӢӨ.
    Thread (мҠӨл Ҳл“ң) н”„лЎңм„ёмҠӨ м•Ҳм—җм„ң мӢӨн–үлҗҳлҠ” кІҪлҹү мӢӨн–ү лӢЁмң„лӢӨ. к°ҷмқҖ н”„лЎңм„ёмҠӨмқҳ мҠӨл Ҳл“ңл“ӨмқҖ Heap, Data, TextлҘј кіөмң н•ҳкі , Stackл§Ң к°Ғмһҗ к°Җм§„лӢӨ. TCBлЎң кҙҖлҰ¬лҗңлӢӨ. л©”лӘЁлҰ¬лҘј кіөмң н•ҳкё° л•Ңл¬ём—җ мғқм„ұ л№„мҡ©мқҙ м Ғкі  лҚ°мқҙн„° көҗнҷҳмқҙ мүҪм§Җл§Ң, лҸҷмӢңм„ұ л¬ём ң(Race Condition)мқҳ мӣҗмқёмқҙ лҗңлӢӨ.
    PCB (Process Control Block) мҡҙмҳҒмІҙм ңк°Җ н”„лЎңм„ёмҠӨлҘј кҙҖлҰ¬н•ҳкё° мң„н•ң мһҗлЈҢкө¬мЎ°лӢӨ. PID, мғҒнғң, л Ҳм§ҖмҠӨн„°, Page Table нҸ¬мқён„°, нҢҢмқј лӘ©лЎқ л“ұмқҙ лӢҙкІЁ мһҲлӢӨ. Context Switch мӢң PCBмқҳ лӮҙмҡ©мқ„ м ҖмһҘн•ҳкі  ліөмӣҗн•ңлӢӨ.
    TCB (Thread Control Block) мҡҙмҳҒмІҙм ңк°Җ мҠӨл Ҳл“ңлҘј кҙҖлҰ¬н•ҳкё° мң„н•ң мһҗлЈҢкө¬мЎ°лӢӨ. Thread ID, л Ҳм§ҖмҠӨн„°, Stack нҸ¬мқён„°к°Җ лӢҙкІЁ мһҲлӢӨ. PCBліҙлӢӨ к°ҖліҚлӢӨ. л©”лӘЁлҰ¬ л§Өн•‘ м •ліҙлҠ” н”„лЎңм„ёмҠӨмқҳ PCBм—җм„ң кіөмң н•ңлӢӨ.
    Memory Layout (л©”лӘЁлҰ¬ л°°м№ҳ) н”„лЎңм„ёмҠӨмқҳ к°ҖмғҒ мЈјмҶҢ кіөк°„мқҙ м–ҙл–»кІҢ кө¬м„ұлҗҳм–ҙ мһҲлҠ”м§ҖлҘј лӮҳнғҖлӮёлӢӨ. Text(мҪ”л“ң), Data(м „м—ӯ ліҖмҲҳ), Heap(лҸҷм Ғ н• лӢ№), Stack(н•ЁмҲҳ нҳём¶ң)мқҳ 4к°ң мҳҒм—ӯмңјлЎң лӮҳлүңлӢӨ.
    Text Segment (н…ҚмҠӨнҠё/мҪ”л“ң мҳҒм—ӯ) н”„лЎңк·ёлһЁмқҳ мӢӨн–ү мҪ”л“ң(кё°кі„м–ҙ)к°Җ м ҖмһҘлҗҳлҠ” мҳҒм—ӯмқҙлӢӨ. Read-only. к°ҷмқҖ н”„лЎңк·ёлһЁмқҳ м—¬лҹ¬ н”„лЎңм„ёмҠӨк°Җ кіөмң н•  мҲҳ мһҲлӢӨ.
    Data Segment (лҚ°мқҙн„° мҳҒм—ӯ) м „м—ӯ ліҖмҲҳмҷҖ static ліҖмҲҳк°Җ м ҖмһҘлҗҳлҠ” мҳҒм—ӯмқҙлӢӨ. н”„лЎңк·ёлһЁ мӢңмһ‘л¶Җн„° лҒқк№Ңм§Җ мЎҙмһ¬н•ңлӢӨ.
    Heap (нһҷ мҳҒм—ӯ) лҸҷм ҒмңјлЎң н• лӢ№лҗҳлҠ” л©”лӘЁлҰ¬ мҳҒм—ӯмқҙлӢӨ. `malloc()`, `new`, Pythonмқҳ к°қмІҙ мғқм„ұ мӢң мӮ¬мҡ©лҗңлӢӨ. м•„лһҳм—җм„ң мң„лЎң мһҗлқјл©°, нҒ¬кё° м ңн•ңмқҙ (кұ°мқҳ) м—ҶлӢӨ. л©”лӘЁлҰ¬ лҲ„мҲҳк°Җ л°ңмғқн•ҳлҠ” кіімқҙлӢӨ.
    Stack (мҠӨнғқ мҳҒм—ӯ) н•ЁмҲҳ нҳём¶ң м •ліҙ(Stack Frame)к°Җ м ҖмһҘлҗҳлҠ” мҳҒм—ӯмқҙлӢӨ. мң„м—җм„ң м•„лһҳлЎң мһҗлқјл©°, нҒ¬кё°к°Җ кі м •лҗҳм–ҙ мһҲлӢӨ (ліҙнҶө 1~8MB). мқҙ нҒ¬кё°лҘј мҙҲкіјн•ҳл©ҙ Stack Overflowк°Җ л°ңмғқн•ңлӢӨ.
    Stack Frame (мҠӨнғқ н”„л Ҳмһ„) н•ЁмҲҳ н•ҳлӮҳк°Җ нҳём¶ңлҗ  л•Ң Stackм—җ мҢ“мқҙлҠ” лҚ°мқҙн„° л¬¶мқҢмқҙлӢӨ. л§Өк°ңліҖмҲҳ, м§Җм—ӯ ліҖмҲҳ, ліөк·Җ мЈјмҶҢк°Җ нҸ¬н•ЁлҗңлӢӨ. мһ¬к·Җ нҳём¶ңл§ҲлӢӨ н•ҳлӮҳм”© м¶”к°ҖлЎң мҢ“мқёлӢӨ.
    Virtual Memory (к°ҖмғҒ л©”лӘЁлҰ¬) OSк°Җ н”„лЎңм„ёмҠӨм—җкІҢ м ңкіөн•ҳлҠ” к°ҖмғҒмқҳ л©”лӘЁлҰ¬ мЈјмҶҢ кіөк°„мқҙлӢӨ. л¬јлҰ¬ л©”лӘЁлҰ¬(RAM)ліҙлӢӨ нҒ° кіөк°„мқ„ м“ё мҲҳ мһҲкІҢ н•ҙмӨҖлӢӨ. Page лӢЁмң„лЎң кҙҖлҰ¬лҗҳл©°, н•„мҡ”н•  л•Ңл§Ң л¬јлҰ¬ л©”лӘЁлҰ¬м—җ л§Өн•‘лҗңлӢӨ.
    Physical Memory (л¬јлҰ¬ л©”лӘЁлҰ¬) мӢӨм ң RAM м№©мқҳ л©”лӘЁлҰ¬лӢӨ. нҒ¬кё°к°Җ л¬јлҰ¬м ҒмңјлЎң кі м •лҗҳм–ҙ мһҲлӢӨ. к°ҖмғҒ мЈјмҶҢлҠ” Page Tableмқ„ нҶөн•ҙ л¬јлҰ¬ мЈјмҶҢлЎң ліҖнҷҳлҗңлӢӨ.
    Page / Page Table (нҺҳмқҙм§Җ / нҺҳмқҙм§Җ н…Ңмқҙлё”) PageлҠ” к°ҖмғҒ л©”лӘЁлҰ¬лҘј мқјм • нҒ¬кё°(ліҙнҶө 4KB)лЎң лӮҳлҲҲ лё”лЎқмқҙлӢӨ. Page TableмқҖ к°ҖмғҒ мЈјмҶҢлҘј л¬јлҰ¬ мЈјмҶҢлЎң ліҖнҷҳн•ҳлҠ” л§Өн•‘ н…Ңмқҙлё”мқҙлӢӨ. н”„лЎңм„ёмҠӨл§ҲлӢӨ лі„лҸ„мқҳ Page Tableмқ„ к°Җм§„лӢӨ.
    Page Fault (нҺҳмқҙм§Җ нҸҙнҠё) м ‘к·јн•ҳл ӨлҠ” Pageк°Җ л¬јлҰ¬ л©”лӘЁлҰ¬м—җ м—Ҷмқ„ л•Ң л°ңмғқн•ҳлҠ” мқён„°лҹҪнҠёлӢӨ. OSк°Җ л””мҠӨнҒ¬(Swap)м—җм„ң н•ҙлӢ№ PageлҘј л¬јлҰ¬ л©”лӘЁлҰ¬лЎң лЎңл“ңн•ңлӢӨ. Mode Switchк°Җ л°ңмғқн•ңлӢӨ.
    OOM (Out of Memory) мӢңмҠӨн…ңмқҳ мӮ¬мҡ© к°ҖлҠҘн•ң л©”лӘЁлҰ¬к°Җ лӘЁл‘җ мҶҢм§„лҗң мғҒнғңлӢӨ. Linuxмқҳ OOM Killerк°Җ н”„лЎңм„ёмҠӨлҘј к°•м ң мў…лЈҢмӢңнӮӨкұ°лӮҳ, Pythonм—җм„ң MemoryErrorк°Җ л°ңмғқн•ңлӢӨ.
    RSS (Resident Set Size) н”„лЎңм„ёмҠӨк°Җ мӢӨм ңлЎң л¬јлҰ¬ л©”лӘЁлҰ¬м—җ мҳ¬л ӨлҶ“кі  мһҲлҠ” лҚ°мқҙн„°мқҳ нҒ¬кё°лӢӨ. "мқҙ н”„лЎңм„ёмҠӨк°Җ RAMмқ„ м–јл§ҲлӮҳ м°Ём§Җн•ҳкі  мһҲлҠ”к°Җ"мқҳ кё°ліё м§Җн‘ңлӢӨ.
    Thrashing (мҠӨлһҳмӢұ) л¬јлҰ¬ л©”лӘЁлҰ¬к°Җ мӢ¬н•ҳкІҢ л¶ҖмЎұн•ҙм„ң, Page In/Outмқҙ лҒҠмһ„м—Ҷмқҙ л°ҳліөлҗҳлҠ” мғҒнғңлӢӨ. CPUлҠ” Page Fault мІҳлҰ¬м—җ лҢҖл¶Җл¶„мқҳ мӢңк°„мқ„ м“°кі  мӢӨм ң мһ‘м—…мқ„ кұ°мқҳ м§„н–үн•ҳм§Җ лӘ»н•ңлӢӨ. н”„лЎңм„ёмҠӨлҘј л„Ҳл¬ҙ л§Һмқҙ лқ„мҡ°л©ҙ Thrashingмқҙ л°ңмғқн•  мҲҳ мһҲлӢӨ.
