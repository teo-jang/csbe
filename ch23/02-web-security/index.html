    Ch.23 мӣ№ ліҙм•Ҳмқҳ н•өмӢ¬¶

    < мӮ¬лЎҖ | мң мӮ¬ мӮ¬лЎҖмҷҖ нӮӨмӣҢл“ң м •лҰ¬ >

    м•һм—җм„ң XSS, SQL Injection, CSRFлҘј лҙӨлӢӨ. м „л¶Җ "мӮ¬мҡ©мһҗ мһ…л Ҙмқ„ мӢ лў°н•ҳм§Җ л§Ҳлқј"м—җм„ң м¶ңл°ңн•ҳлҠ” кіөкІ©мқҙм—ҲлӢӨ. мқҙлІҲм—җлҠ” лІ”мң„лҘј л„“нҳҖм„ң, мӣ№ ліҙм•Ҳмқҳ лӢӨлҘё н•өмӢ¬ мЈјм ңл“Өмқ„ лӢӨлЈ¬лӢӨ. лёҢлқјмҡ°м Җмқҳ ліҙм•Ҳ м •мұ…, нҶөмӢ  м•”нҳёнҷ”, мқёмҰқ л°©мӢқ, л№„л°ҖлІҲнҳё кҙҖлҰ¬, л¬ҙм°Ёлі„ лҢҖмһ… л°©м–ҙк№Ңм§Җ.

    Same-Origin PolicyмҷҖ CORS¶

    "CORS м—җлҹ¬"лҘј н•ң лІҲлҸ„ м•Ҳ ліё к°ңл°ңмһҗлҠ” м—Ҷмқ„ кұ°лӢӨ.

    Access to fetch at 'https://api.example.com/data' from origin
'https://app.example.com' has been blocked by CORS policy

    н”„лЎ нҠём—”л“ңм—җм„ң л°ұм—”л“ң APIлҘј нҳём¶ңн–ҲлҚ”лӢҲ мқҙ м—җлҹ¬к°Җ лң¬лӢӨ. "мҷң м•Ҳ лҗҳлҠ” кұ°м§Җ?"лқјкі  кө¬кёҖл§Ғн•ҳл©ҙ "CORS н—ӨлҚ”лҘј м¶”к°Җн•ҙлқј"лқјлҠ” лӢөмқҙ лӮҳмҳЁлӢӨ. н—ӨлҚ”лҘј м¶”к°Җн•ҳл©ҙ лҗңлӢӨ. к·ёлҹ°лҚ° мҷң мқҙлҹ° м •мұ…мқҙ мһҲлҠ”к°Җ?

    Same-Origin Policy (лҸҷмқј м¶ңмІҳ м •мұ…) лёҢлқјмҡ°м Җмқҳ ліҙм•Ҳ м •мұ…мқҙлӢӨ. "к°ҷмқҖ м¶ңмІҳ(Origin)м—җм„ң мҳЁ лҰ¬мҶҢмҠӨл§Ң м ‘к·јн•  мҲҳ мһҲлӢӨ"лҠ” к·ңм№ҷмқҙлӢӨ. м—¬кё°м„ң OriginмқҖ н”„лЎңнҶ мҪң + лҸ„л©”мқё + нҸ¬нҠёмқҳ мЎ°н•©мқҙлӢӨ. - `https://app.example.com` кіј `https://api.example.com` вҶ’ лҸ„л©”мқёмқҙ лӢӨлҘҙлӢҲ лӢӨлҘё Origin - `https://example.com` кіј `http://example.com` вҶ’ н”„лЎңнҶ мҪңмқҙ лӢӨлҘҙлӢҲ лӢӨлҘё Origin - `https://example.com:443` кіј `https://example.com:8080` вҶ’ нҸ¬нҠёк°Җ лӢӨлҘҙлӢҲ лӢӨлҘё Origin мқҙ м •мұ…мқҙ м—Ҷмңјл©ҙ? м•…м„ұ мӮ¬мқҙнҠё evil.comм—җ м ‘мҶҚн–Ҳмқ„ л•Ң, evil.comмқҳ мһҗл°”мҠӨнҒ¬лҰҪнҠёк°Җ bank.comмқҳ APIлҘј нҳём¶ңн•ҙм„ң мһ”кі лҘј мЎ°нҡҢн•ҳкұ°лӮҳ мҶЎкёҲмқ„ н•  мҲҳ мһҲлӢӨ. мӮ¬мҡ©мһҗк°Җ bank.comм—җ лЎңк·ёмқён•ң мғҒнғңлқјл©ҙ мҝ нӮӨк°Җ мһҗлҸҷмңјлЎң нҸ¬н•ЁлҗҳлӢҲк№Ң. Same-Origin Policyк°Җ мқҙкұё л§үлҠ”лӢӨ.

    Same-Origin PolicyлҠ” ліҙм•Ҳмқ„ мң„н•ҙ мЎҙмһ¬н•ңлӢӨ. к·ёлҹ°лҚ° нҳ„мӢӨм—җм„ңлҠ” н”„лЎ нҠём—”л“ң(app.example.com)мҷҖ л°ұм—”л“ң(api.example.com)к°Җ лӢӨлҘё лҸ„л©”мқём—җ мһҲлҠ” кІҪмҡ°к°Җ л§ҺлӢӨ. мқҙл•Ң "мқҙ лҸ„л©”мқёмқҳ мҡ”мІӯмқҖ н—Ҳмҡ©н•ҙлҸ„ лҗңлӢӨ"кі  м„ңлІ„к°Җ лӘ…мӢңм ҒмңјлЎң м„ м–ён•ҳлҠ” кІҢ CORSлӢӨ.

    CORS (Cross-Origin Resource Sharing, көҗм°Ё м¶ңмІҳ лҰ¬мҶҢмҠӨ кіөмң ) Same-Origin PolicyлҘј м„ нғқм ҒмңјлЎң мҷ„нҷ”н•ҳлҠ” л©”м»ӨлӢҲмҰҳмқҙлӢӨ. м„ңлІ„к°Җ HTTP мқ‘лӢө н—ӨлҚ”м—җ "мқҙ Originм—җм„ң мҳӨлҠ” мҡ”мІӯмқҖ н—Ҳмҡ©н•ңлӢӨ"кі  лӘ…мӢңн•ҳл©ҙ, лёҢлқјмҡ°м Җк°Җ н•ҙлӢ№ мҡ”мІӯмқ„ н—Ҳмҡ©н•ңлӢӨ. н•өмӢ¬ н—ӨлҚ”: - `Access-Control-Allow-Origin`: н—Ҳмҡ©н•  Origin (мҳҲ: `https://app.example.com` лҳҗлҠ” мҷҖмқјл“ңм№ҙл“ң `*`) - `Access-Control-Allow-Methods`: н—Ҳмҡ©н•  HTTP л©”м„ңл“ң (GET, POST л“ұ) - `Access-Control-Allow-Headers`: н—Ҳмҡ©н•  м»ӨмҠӨн…Җ н—ӨлҚ” - `Access-Control-Allow-Credentials`: мҝ нӮӨ нҸ¬н•Ё м—¬л¶Җ (true/false)

    FastAPIм—җм„ң CORSлҘј м„Өм •н•ҳлҠ” мҪ”л“ң:

    from fastapi.middleware.cors import CORSMiddleware

app.add_middleware(
    CORSMiddleware,
    allow_origins=["https://app.example.com"],  # нҠ№м • лҸ„л©”мқёл§Ң н—Ҳмҡ©
    allow_credentials=True,
    allow_methods=["GET", "POST"],
    allow_headers=["*"],
)

    м—¬кё°м„ң нқ”н•ң мӢӨмҲҳ:

    # мқҙл ҮкІҢ н•ҳл©ҙ лӘЁл“  Originмқ„ н—Ҳмҡ©н•ңлӢӨ - к°ңл°ң нҷҳкІҪм—җм„ңл§Ң мҚЁм•ј н•ңлӢӨ
allow_origins=["*"]

    allow_origins=["*"]м—җ allow_credentials=TrueлҘј к°ҷмқҙ м“°л©ҙ? лёҢлқјмҡ°м Җк°Җ кұ°л¶Җн•ңлӢӨ. лӘЁл“  Originмқ„ н—Ҳмҡ©н•ҳл©ҙм„ң мҝ нӮӨк№Ңм§Җ ліҙлӮҙкІҢ н•ҳл©ҙ Same-Origin PolicyлҘј л¬ҙл Ҙнҷ”н•ҳлҠ” кІғмқҙлӮҳ л§Ҳм°¬к°Җм§Җмқҙкё° л•Ңл¬ёмқҙлӢӨ.

    CORSлҠ” лёҢлқјмҡ°м Җмқҳ м •мұ…мқҙлӢӨ. м„ңлІ„ к°„ нҶөмӢ (м„ңлІ„м—җм„ң лӢӨлҘё м„ңлІ„лЎң мҡ”мІӯмқ„ ліҙлӮҙлҠ” кІҪмҡ°)м—җлҠ” CORSк°Җ м Ғмҡ©лҗҳм§Җ м•ҠлҠ”лӢӨ. curlмқҙлӮҳ PostmanмңјлЎң мҡ”мІӯн•ҳл©ҙ CORS м—җлҹ¬к°Җ м•Ҳ лӮҳлҠ” мқҙмң к°Җ мқҙкұ°лӢӨ.

    HTTPSмҷҖ TLS: нҶөмӢ  м•”нҳёнҷ”¶

    HTTPлЎң лҚ°мқҙн„°лҘј ліҙлӮҙл©ҙ м–ҙл–»кІҢ лҗҳлҠ”к°Җ?

    [лёҢлқјмҡ°м Җ] ---"password=1234"---> [м„ңлІ„]

    мқҙ лҚ°мқҙн„°к°Җ л„ӨнҠёмӣҢнҒ¬лҘј м§ҖлӮҳк°ҖлҠ” лҸҷм•Ҳ, мӨ‘к°„м—җ мһҲлҠ” лҲ„кө¬л“  ліј мҲҳ мһҲлӢӨ. к°ҷмқҖ Wi-FiлҘј м“°лҠ” мӮ¬лһҢ, ISP(мқён„°л„· м„ңл№„мҠӨ м ңкіөмһҗ), мӨ‘к°„ лқјмҡ°н„° кҙҖлҰ¬мһҗ. HTTPлҠ” нҸүл¬ё(plaintext)мқҙлӢӨ. лҸ„мІӯмқҙ к°ҖлҠҘн•ҳлӢӨ.

    HTTPSлҠ” HTTPм—җ TLSлҘј м”Ңмҡҙ кІғмқҙлӢӨ. TLSк°Җ нҶөмӢ мқ„ м•”нҳёнҷ”н•ҙм„ң, мӨ‘к°„м—җм„ң лҙҗлҸ„ лӮҙмҡ©мқ„ м•Ң мҲҳ м—ҶкІҢ л§Ңл“ лӢӨ.

    HTTPS / TLS (Transport Layer Security) HTTPSлҠ” HTTP + TLSлӢӨ. TLSлҠ” нҶөмӢ  м•”нҳёнҷ” н”„лЎңнҶ мҪңлЎң, Ch.6м—җм„ң лӢӨлӨҳлҚҳ TCP мң„м—җм„ң лҸҷмһ‘н•ңлӢӨ. TCP 3-Way Handshake нӣ„, TLS Handshakeк°Җ м¶”к°ҖлЎң м§„н–үлҗңлӢӨ. TLSк°Җ ліҙмһҘн•ҳлҠ” кІғ м„ё к°Җм§Җ: - кё°л°Җм„ұ(Confidentiality): лҚ°мқҙн„°к°Җ м•”нҳёнҷ”лҗҳм–ҙ лҸ„мІӯн•  мҲҳ м—ҶлӢӨ - л¬ҙкІ°м„ұ(Integrity): лҚ°мқҙн„°к°Җ мӨ‘к°„м—җ ліҖмЎ°лҗҳл©ҙ к°җм§ҖлҗңлӢӨ - мқёмҰқ(Authentication): м„ңлІ„к°Җ м§„м§ң к·ё м„ңлІ„мқём§Җ мқёмҰқм„ңлЎң нҷ•мқён•ңлӢӨ TLS 1.3мқҙ нҳ„мһ¬ мөңмӢ  лІ„м „мқҙлӢӨ. TLS 1.2 мқҙм „ лІ„м „(SSL 3.0, TLS 1.0, TLS 1.1)мқҖ м•Ңл Өм§„ м·Ём•Ҫм җмқҙ мһҲм–ҙм„ң мӮ¬мҡ©н•ҳл©ҙ м•Ҳ лҗңлӢӨ.

    TLSм—җм„ң м•”нҳёнҷ”к°Җ м–ҙл–»кІҢ мқҙлЈЁм–ҙм§ҖлҠ”м§Җ к°„лһөнһҲ ліҙл©ҙ:

    лҢҖм№ӯнӮӨ м•”нҳёнҷ” (Symmetric Encryption) к°ҷмқҖ нӮӨлЎң м•”нҳёнҷ”н•ҳкі  ліөнҳёнҷ”н•ҳлҠ” л°©мӢқмқҙлӢӨ. л№ лҘҙлӢӨ. н•ҳм§Җл§Ң "нӮӨлҘј м–ҙл–»кІҢ м•Ҳм „н•ҳкІҢ м „лӢ¬н•ҳлҠ”к°Җ"к°Җ л¬ём ңлӢӨ. нӮӨлҘј л„ӨнҠёмӣҢнҒ¬лЎң ліҙлӮҙлҠ” мҲңк°„ лҸ„мІӯлҗ  мҲҳ мһҲлӢӨ. AESк°Җ лҢҖн‘ңм Ғмқё лҢҖм№ӯнӮӨ м•Ңкі лҰ¬мҰҳмқҙлӢӨ.
    л№„лҢҖм№ӯнӮӨ м•”нҳёнҷ” (Asymmetric Encryption) кіөк°ңнӮӨ(Public Key)мҷҖ к°ңмқёнӮӨ(Private Key) мҢҚмқ„ мӮ¬мҡ©н•ңлӢӨ. кіөк°ңнӮӨлЎң м•”нҳёнҷ”н•ң лҚ°мқҙн„°лҠ” к°ңмқёнӮӨлЎңл§Ң ліөнҳёнҷ”н•  мҲҳ мһҲлӢӨ. лҠҗлҰ¬м§Җл§Ң, нӮӨ көҗнҷҳ л¬ём ңлҘј н•ҙкІ°н•ңлӢӨ. кіөк°ңнӮӨлҠ” л§җ к·ёлҢҖлЎң кіөк°ңн•ҙлҸ„ лҗҳлӢҲк№Ң. RSA, ECDHEк°Җ лҢҖн‘ңм ҒмқҙлӢӨ.

    TLSлҠ” мқҙ л‘ҳмқ„ мЎ°н•©н•ҙм„ң м“ҙлӢӨ. мІҳмқҢм—җ л№„лҢҖм№ӯнӮӨлЎң "м„ём…ҳ нӮӨ(лҢҖм№ӯнӮӨ)"лҘј м•Ҳм „н•ҳкІҢ көҗнҷҳн•ҳкі , мқҙнӣ„ нҶөмӢ мқҖ л№ лҘё лҢҖм№ӯнӮӨлЎң м•”нҳёнҷ”н•ңлӢӨ. л№„лҢҖм№ӯнӮӨмқҳ м•Ҳм „м„ұкіј лҢҖм№ӯнӮӨмқҳ мҶҚлҸ„лҘј л‘ҳ лӢӨ м·Ён•ҳлҠ” кө¬мЎ°лӢӨ.

    sequenceDiagram
    participant C as Client (лёҢлқјмҡ°м Җ)
    participant S as Server

    Note over C,S: TCP 3-Way Handshake (Ch.6)
    C->>S: ClientHello (м§Җмӣҗн•ҳлҠ” м•”нҳёнҷ” лӘ©лЎқ)
    S->>C: ServerHello + мқёмҰқм„ң (кіөк°ңнӮӨ нҸ¬н•Ё)
    Note left of C: мқёмҰқм„ң кІҖмҰқ (CA м„ңлӘ… нҷ•мқё)
    C->>S: м„ём…ҳ нӮӨ көҗнҷҳ (л№„лҢҖм№ӯнӮӨлЎң м•”нҳёнҷ”)
    Note over C,S: TLS Handshake мҷ„лЈҢ
    C->>S: м•”нҳёнҷ”лҗң HTTP мҡ”мІӯ (лҢҖм№ӯнӮӨ)
    S->>C: м•”нҳёнҷ”лҗң HTTP мқ‘лӢө (лҢҖм№ӯнӮӨ)

    "HTTPSлҠ” лҠҗлҰ¬лӢӨ"лҠ” л§җмқ„ л“Өм–ҙліё м Ғмқҙ мһҲмқ„ мҲҳ мһҲлӢӨ. TLS Handshake л•Ңл¬ём—җ мөңмҙҲ м—°кІ° мӢң м•Ҫк°„мқҳ м§Җм—°мқҙ м¶”к°ҖлҗҳлҠ” кұҙ л§һлӢӨ. н•ҳм§Җл§Ң TLS 1.3м—җм„ңлҠ” Handshakeк°Җ 1-RTT(кё°мЎҙ 1.2лҠ” 2-RTT)лЎң мӨ„м—Ҳкі , 0-RTT мһ¬м—°кІ°лҸ„ м§Җмӣҗн•ңлӢӨ. нҳ„лҢҖ н•ҳл“ңмӣЁм–ҙм—җм„ң AES м•”нҳёнҷ”лҠ” CPU лӘ…л №м–ҙ мҲҳмӨҖмңјлЎң м§Җмӣҗлҗҳм–ҙ(AES-NI) м„ұлҠҘ л¶ҖлӢҙмқҙ кұ°мқҳ м—ҶлӢӨ. "лҠҗлҰ¬лӢҲк№Ң HTTPлҘј м“°кІ лӢӨ"лҠ” мқҙмң к°Җ лҗ  мҲҳ м—ҶлӢӨ.

    2024л…„ кё°мӨҖ, м „мІҙ мӣ№ нҠёлһҳн”Ҫмқҳ м•Ҫ 95% мқҙмғҒмқҙ HTTPSлҘј мӮ¬мҡ©н•ңлӢӨ. ChromeмқҖ HTTP мӮ¬мқҙнҠём—җ "мЈјмқҳ мҡ”н•Ё" кІҪкі лҘј н‘ңмӢңн•ңлӢӨ. Let's Encrypt к°ҷмқҖ м„ңл№„мҠӨлЎң л¬ҙлЈҢ мқёмҰқм„ңлҘј л°ңкёүл°ӣмқ„ мҲҳ мһҲлӢӨ. HTTPSлҘј м•Ҳ м“ё мқҙмң к°Җ м—ҶлӢӨ.

    (м¶ңмІҳ: Google Transparency Report, HTTPS encryption on the web, 2024)

    мқёмҰқ л°©мӢқ: Session vs JWT¶

    мӮ¬мҡ©мһҗк°Җ лЎңк·ёмқён•ҳл©ҙ, мқҙнӣ„ мҡ”мІӯм—җм„ң "мқҙ мӮ¬мҡ©мһҗлҠ” мқёмҰқлҗң мӮ¬мҡ©мһҗлӢӨ"лқјлҠ” кұё м„ңлІ„к°Җ м–ҙл–»кІҢ м•Ң мҲҳ мһҲлҠ”к°Җ? лҢҖн‘ңм ҒмңјлЎң Session л°©мӢқкіј JWT л°©мӢқмқҙ мһҲлӢӨ.

    Session кё°л°ҳ мқёмҰқ¶

    Session (м„ём…ҳ) м„ңлІ„м—җ мӮ¬мҡ©мһҗ мғҒнғңлҘј м ҖмһҘн•ҳлҠ” мқёмҰқ л°©мӢқмқҙлӢӨ. лЎңк·ёмқён•ҳл©ҙ м„ңлІ„к°Җ м„ём…ҳ IDлҘј мғқм„ұн•ҳкі , мқҙ IDлҘј мҝ нӮӨлЎң лёҢлқјмҡ°м Җм—җ ліҙлӮёлӢӨ. мқҙнӣ„ мҡ”мІӯм—җм„ң лёҢлқјмҡ°м Җк°Җ мҝ нӮӨлҘј мһҗлҸҷмңјлЎң м „мҶЎн•ҳл©ҙ, м„ңлІ„к°Җ м„ём…ҳ IDлЎң мӮ¬мҡ©мһҗлҘј мӢқлі„н•ңлӢӨ. м„ём…ҳ лҚ°мқҙн„°(мӮ¬мҡ©мһҗ м •ліҙ, к¶Ңн•ң л“ұ)лҠ” м„ңлІ„мқҳ л©”лӘЁлҰ¬лӮҳ Redisм—җ м ҖмһҘлҗңлӢӨ. (Java Springмқҳ HttpSession, Djangoмқҳ session frameworkмқҙ мқҙ л°©мӢқмқҙлӢӨ.) 
    sequenceDiagram
    participant B as лёҢлқјмҡ°м Җ
    participant S as м„ңлІ„
    participant R as м„ём…ҳ м ҖмһҘмҶҢ (Redis)

    B->>S: POST /login (id, password)
    S->>R: м„ём…ҳ мғқм„ұ (session_id: abc123, user: teo)
    S->>B: Set-Cookie: session_id=abc123
    Note over B: мҝ нӮӨ м ҖмһҘ

    B->>S: GET /profile (Cookie: session_id=abc123)
    S->>R: м„ём…ҳ мЎ°нҡҢ (abc123 вҶ’ user: teo)
    S->>B: 200 OK (teoмқҳ н”„лЎңн•„)

    мһҘм җ:

    • м„ңлІ„к°Җ м„ём…ҳмқ„ мҷ„м „нһҲ м ңм–ҙн•ңлӢӨ. к°•м ң лЎңк·ём•„мӣғ(м„ём…ҳ мӮӯм ң)мқҙ мҰүмӢң к°ҖлҠҘн•ҳлӢӨ.
    • мҝ нӮӨм—җ HttpOnly, Secure, SameSite мҶҚм„ұмқ„ м„Өм •н•ҳл©ҙ XSS/CSRF л°©м–ҙк°Җ к°ҖлҠҘн•ҳлӢӨ.
    • м„ём…ҳ лҚ°мқҙн„°к°Җ м„ңлІ„м—җл§Ң мһҲмңјлӢҲ, нҒҙлқјмқҙм–ёнҠём—җ лҜјк°җ м •ліҙк°Җ л…ём¶ңлҗҳм§Җ м•ҠлҠ”лӢӨ.

    лӢЁм җ:

    • м„ңлІ„к°Җ мғҒнғңлҘј м ҖмһҘн•ҙм•ј н•ңлӢӨ (Stateful). м„ңлІ„к°Җ м—¬лҹ¬ лҢҖл©ҙ м„ём…ҳ кіөмң к°Җ н•„мҡ”н•ҳлӢӨ (Redis л“ұ).
    • л§Ө мҡ”мІӯл§ҲлӢӨ м„ём…ҳ м ҖмһҘмҶҢлҘј мЎ°нҡҢн•ҙм•ј н•ңлӢӨ.

    JWT кё°л°ҳ мқёмҰқ¶

    JWT (JSON Web Token) м„ңлІ„к°Җ л°ңкёүн•ҳлҠ” м„ңлӘ…лҗң нҶ нҒ°мқҙлӢӨ. нҶ нҒ° мһҗмІҙм—җ мӮ¬мҡ©мһҗ м •ліҙ(payload)к°Җ нҸ¬н•Ёлҗҳм–ҙ мһҲкі , м„ңлІ„мқҳ л№„л°ҖнӮӨлЎң м„ңлӘ…лҗҳм–ҙ мһҲлӢӨ. м„ңлІ„лҠ” нҶ нҒ°мқҳ м„ңлӘ…л§Ң кІҖмҰқн•ҳл©ҙ лҗҳлҜҖлЎң, лі„лҸ„мқҳ м„ём…ҳ м ҖмһҘмҶҢк°Җ н•„мҡ” м—ҶлӢӨ (Stateless). кө¬мЎ°: Header.Payload.Signature (Base64 мқёмҪ”л”©, м җмңјлЎң кө¬л¶„) - Header: м•Ңкі лҰ¬мҰҳ м •ліҙ (мҳҲ: HS256) - Payload: мӮ¬мҡ©мһҗ м •ліҙ (claims). мҳҲ: `{"user_id": 123, "role": "admin", "exp": 1234567890}` - Signature: Header + PayloadлҘј м„ңлІ„ л№„л°ҖнӮӨлЎң м„ңлӘ…н•ң к°’ мЈјмқҳ: JWTмқҳ PayloadлҠ” м•”нҳёнҷ”к°Җ м•„лӢҲлқј Base64 мқёмҪ”л”©мқҙлӢӨ. лҲ„кө¬лӮҳ л””мҪ”л”©н•ҙм„ң лӮҙмҡ©мқ„ ліј мҲҳ мһҲлӢӨ. лҜјк°җн•ң м •ліҙ(л№„л°ҖлІҲнҳё л“ұ)лҘј л„Јмңјл©ҙ м•Ҳ лҗңлӢӨ. 
    sequenceDiagram
    participant B as лёҢлқјмҡ°м Җ
    participant S as м„ңлІ„

    B->>S: POST /login (id, password)
    Note right of S: JWT мғқм„ұ + м„ңлӘ…
    S->>B: {"token": "eyJhbG..."}
    Note over B: нҶ нҒ° м ҖмһҘ (localStorage лҳҗлҠ” мҝ нӮӨ)

    B->>S: GET /profile (Authorization: Bearer eyJhbG...)
    Note right of S: м„ңлӘ… кІҖмҰқл§Ң н•ҳл©ҙ лҗЁ (DB мЎ°нҡҢ л¶Ҳн•„мҡ”)
    S->>B: 200 OK (teoмқҳ н”„лЎңн•„)

    мһҘм җ:

    • м„ңлІ„к°Җ мғҒнғңлҘј м ҖмһҘн•ҳм§Җ м•ҠлҠ”лӢӨ (Stateless). м„ңлІ„лҘј м•„л¬ҙлҰ¬ лҠҳл ӨлҸ„ м„ём…ҳ кіөмң  л¬ём ңк°Җ м—ҶлӢӨ.
    • м„ём…ҳ м ҖмһҘмҶҢ мЎ°нҡҢк°Җ м—ҶмңјлӢҲ л№ лҘҙлӢӨ.

    лӢЁм җ:

    • нҶ нҒ° нғҲм·Ё мӢң л§ҢлЈҢлҗ  л•Ңк№Ңм§Җ л¬ҙнҡЁнҷ”н•  мҲҳ м—ҶлӢӨ. м„ңлІ„м—җ "мқҙ нҶ нҒ°мқҖ л¬ҙнҡЁ"лқјлҠ” м •ліҙлҘј м ҖмһҘн•ҳл©ҙ? к·ё мҲңк°„ Statelessмқҳ мһҘм җмқҙ мӮ¬лқјм§„лӢӨ.
    • нҶ нҒ°м—җ мӮ¬мҡ©мһҗ м •ліҙк°Җ нҸ¬н•Ёлҗҳм–ҙ мһҲмңјлҜҖлЎң, к¶Ңн•ң ліҖкІҪмқҙ мҰүмӢң л°ҳмҳҒлҗҳм§Җ м•ҠлҠ”лӢӨ.
    • нҶ нҒ° нҒ¬кё°к°Җ м„ём…ҳ IDліҙлӢӨ нҒ¬лӢӨ (л§Ө мҡ”мІӯм—җ нҸ¬н•ЁлҗЁ).

    м–ҙлҠҗ кІҢ мўӢмқҖк°Җ?¶

    "мғҒнҷ©м—җ л”°лқј лӢӨлҘҙлӢӨ"к°Җ м •лӢөмқҙкі , мқҙкұҙ ліҖлӘ…мқҙ м•„лӢҲлӢӨ.

    мғҒнҷ© м¶”мІң мқҙмң 
    лӢЁмқј м„ңлІ„, м „нҶөм Ғ мӣ№м•ұ Session к°„лӢЁн•ҳкі , к°•м ң лЎңк·ём•„мӣғмқҙ мүҪлӢӨ
    MSA, м„ңлІ„ м—¬лҹ¬ лҢҖ JWT м„ём…ҳ кіөмң  мқён”„лқј л¶Ҳн•„мҡ”
    лӘЁл°”мқј м•ұ + API JWT мҝ нӮӨ лҢҖмӢ  HeaderлЎң м „мҶЎ
    ліҙм•Ҳмқҙ к·№лҸ„лЎң мӨ‘мҡ” (кёҲмңө) Session + Redis мҰүмӢң л¬ҙнҡЁнҷ”к°Җ к°ҖлҠҘн•ҙм•ј н•ңлӢӨ

    мӢӨл¬ҙм—җм„ңлҠ” Access Token(JWT, м§§мқҖ л§ҢлЈҢ) + Refresh Token(кёҙ л§ҢлЈҢ, м„ңлІ„ м ҖмһҘ) мЎ°н•©мқ„ л§Һмқҙ м“ҙлӢӨ. Access Tokenмқҙ нғҲм·ЁлҸјлҸ„ л§ҢлЈҢк°Җ м§§м•„м„ң н”јн•ҙк°Җ м ңн•ңлҗҳкі , Refresh TokenмңјлЎң мғҲ Access Tokenмқ„ л°ңкёүл°ӣлҠ” кө¬мЎ°лӢӨ.

    л№„л°ҖлІҲнҳё кҙҖлҰ¬: нҸүл¬ё м ҖмһҘмқҖ лІ”мЈ„лӢӨ¶

    "м„Өл§Ҳ л№„л°ҖлІҲнҳёлҘј нҸүл¬ёмңјлЎң м ҖмһҘн•ҳлҠ” м„ңл№„мҠӨк°Җ мһҲкІ м–ҙ?"лқјкі  мғқк°Ғн•  мҲҳ мһҲлӢӨ. мһҲлӢӨ. 2019л…„ Facebookмқҙ мҲҳм–ө к°ңмқҳ мӮ¬мҡ©мһҗ л№„л°ҖлІҲнҳёлҘј лӮҙл¶Җ лЎңк·ём—җ нҸүл¬ёмңјлЎң м ҖмһҘн•ҳкі  мһҲм—ҲлӢӨлҠ” мӮ¬мӢӨмқҙ л°қнҳҖмЎҢлӢӨ.

    (м¶ңмІҳ: KrebsOnSecurity, "Facebook Stored Hundreds of Millions of User Passwords in Plain Text for Years", 2019)

    л№„л°ҖлІҲнҳёлҠ” л°ҳл“ңмӢң н•ҙмӢң(hash)н•ҙм„ң м ҖмһҘн•ҙм•ј н•ңлӢӨ.

    # м ҲлҢҖ н•ҳл©ҙ м•Ҳ лҗҳлҠ” кІғ
db.execute("INSERT INTO users (password) VALUES (:pw)", {"pw": "1234"})

# н•ҙм•ј н•ҳлҠ” кІғ
import bcrypt

hashed = bcrypt.hashpw("1234".encode(), bcrypt.gensalt())
db.execute("INSERT INTO users (password) VALUES (:pw)", {"pw": hashed})

    мҷң лӢЁмҲң н•ҙмӢң(SHA-256 к°ҷмқҖ)к°Җ м•„лӢҲлқј bcryptлҘј м“°лҠ”к°Җ? SHA-256мқҖ л№ лҘҙлӢӨ. л№ лҘҙлӢӨлҠ” кұҙ кіөкІ©мһҗк°Җ 1мҙҲм—җ мҲҳмӢӯм–ө к°ңмқҳ н•ҙмӢңлҘј мӢңлҸ„н•  мҲҳ мһҲлӢӨлҠ” лң»мқҙлӢӨ. bcrypt, scrypt, argon2 к°ҷмқҖ "лҠҗлҰ° н•ҙмӢң н•ЁмҲҳ"лҠ” мқҳлҸ„м ҒмңјлЎң м—°мӮ°мқ„ лҠҗлҰ¬кІҢ л§Ңл“Өм–ҙм„ң л¬ҙм°Ёлі„ лҢҖмһ…(Brute Force)мқ„ л№„мӢӨмҡ©м ҒмңјлЎң л§Ңл“ лӢӨ.

    н•ҙмӢң н•ЁмҲҳ мҙҲлӢ№ мӢңлҸ„ нҡҹмҲҳ (GPU) мҡ©лҸ„
    SHA-256 мҲҳмӢӯм–ө лҚ°мқҙн„° л¬ҙкІ°м„ұ кІҖмҰқ (л№„л°ҖлІҲнҳём—җ л¶Җм Ғн•©)
    bcrypt мҲҳл§Ң л№„л°ҖлІҲнҳё н•ҙмӢұ
    argon2 мҲҳмІң л№„л°ҖлІҲнҳё н•ҙмӢұ (л©”лӘЁлҰ¬лҸ„ л§Һмқҙ мӮ¬мҡ©)

    (bcryptмқҳ cost factorлҘј мҳ¬лҰ¬л©ҙ лҚ” лҠҗл Өм§„лӢӨ. кё°ліёк°’мқҖ 12лЎң, н•ҳлӮҳмқҳ н•ҙмӢңм—җ м•Ҫ 200~300msк°Җ кұёлҰ°лӢӨ. кіөкІ©мһҗм—җкІҢлҠ” мқҙ 200msк°Җ мҲҳмӢӯм–ө лІҲ кіұн•ҙм§„лӢӨ.)

    SaltлҠ” к°ҷмқҖ л№„л°ҖлІҲнҳёлқјлҸ„ лӢӨлҘё н•ҙмӢңк°’мқ„ л§Ңл“Өм–ҙмӨҖлӢӨ. "1234"лҘј 10лӘ…мқҙ м“°лҚ”лқјлҸ„ н•ҙмӢңк°’мқҙ м „л¶Җ лӢӨлҘҙлӢӨ. Rainbow Table(лҜёлҰ¬ кі„мӮ°н•ҙл‘” н•ҙмӢң-л№„л°ҖлІҲнҳё лҢҖмқ‘н‘ң) кіөкІ©мқ„ л°©м–ҙн•ңлӢӨ. bcryptлҠ” SaltлҘј мһҗлҸҷмңјлЎң нҸ¬н•Ён•ңлӢӨ.

    Rate Limiting: л¬ҙм°Ёлі„ лҢҖмһ… л°©м–ҙ¶

    л№„л°ҖлІҲнҳёлҘј м•Ҳм „н•ҳкІҢ м ҖмһҘн•ҳлҠ” кІғл§ҢмңјлЎңлҠ” л¶ҖмЎұн•ҳлӢӨ. кіөкІ©мһҗк°Җ лЎңк·ёмқё APIм—җ л№„л°ҖлІҲнҳёлҘј л¬ҙн•ңнһҲ мӢңлҸ„н•  мҲҳ мһҲмңјл©ҙ, мӢңк°„мқҙ м¶©л¶„н•ҳл©ҙ кІ°көӯ лҡ«лҰ°лӢӨ.

    Rate LimitingмқҖ мқјм • мӢңк°„ лӮҙ мҡ”мІӯ нҡҹмҲҳлҘј м ңн•ңн•ҳлҠ” кё°лІ•мқҙлӢӨ.

    from fastapi import Request
from collections import defaultdict
import time

# к°„лӢЁн•ң Rate Limiter (мӢӨл¬ҙм—җм„ңлҠ” Redis кё°л°ҳмңјлЎң кө¬нҳ„)
_login_attempts = defaultdict(list)

@router.post("/login")
def login(request: Request, username: str, password: str):
    client_ip = request.client.host
    now = time.time()

    # мөңк·ј 5л¶„к°„мқҳ мӢңлҸ„ нҡҹмҲҳ нҷ•мқё
    recent = [t for t in _login_attempts[client_ip] if now - t < 300]
    _login_attempts[client_ip] = recent

    if len(recent) >= 5:
        raise HTTPException(
            status_code=429,
            detail="Too many login attempts. Try again in 5 minutes."
        )

    _login_attempts[client_ip].append(now)

    # лЎңк·ёмқё кІҖмҰқ лЎңм§Ғ...

    5л¶„м—җ 5лІҲк№Ңм§Җл§Ң н—Ҳмҡ©н•ңлӢӨ. 6лІҲм§ёл¶Җн„°лҠ” 429(Too Many Requests)лҘј л°ҳнҷҳн•ңлӢӨ. л¬ҙм°Ёлі„ лҢҖмһ… кіөкІ©мқҙ л№„мӢӨмҡ©м Ғмқҙ лҗңлӢӨ.

    мӢӨл¬ҙм—җм„ңлҠ” nginxлӮҳ API Gateway л ҲлІЁм—җм„ң Rate Limitingмқ„ кұ°лҠ” кІҪмҡ°к°Җ л§ҺлӢӨ. м• н”ҢлҰ¬мјҖмқҙм…ҳ мҪ”л“ңк№Ңм§Җ лҸ„лӢ¬н•ҳкё° м „м—җ м°ЁлӢЁн•ҳлҠ” кІҢ нҡЁмңЁм ҒмқҙлӢӨ. Redisмқҳ INCR + EXPIRE мЎ°н•©мңјлЎң л¶„мӮ° нҷҳкІҪм—җм„ңлҸ„ мқјкҙҖлҗң Rate Limitingмқ„ кө¬нҳ„н•  мҲҳ мһҲлӢӨ.

    XSSлҠ” HTMLм—җ мӮ¬мҡ©мһҗ мһ…л Ҙмқ„ л„Јкё° м „м—җ escapeн•ҳл©ҙ л§үнһҲкі , SQL InjectionмқҖ Parameterized QueryлҘј м“°л©ҙ л§үнһҲкі , CSRFлҠ” нҶ нҒ°мңјлЎң м¶ңмІҳлҘј кІҖмҰқн•ҳл©ҙ л§үнһҢлӢӨ.
