# Ch.7 AI가 코드를 짜주는 시대, 왜 CS를 알아야 하는가

Part 1에서 System Call, CPU/I/O Bound, 메모리 구조, 동시성, 네트워크까지 훑었다. "이걸 왜 알아야 하는가"에 대한 답은 Ch.1에서 했다. 키워드를 모르면 검색도 못 하고, 면접에서 대답도 못 한다고.

그런데 GPT-4가 나온 이후로 상황이 하나 더 추가됐다. AI가 코드를 짜주는 시대가 됐다. 그리고 이 AI에게 지시를 내리는 것도, 결국 키워드가 결정한다.

## 1. 환경 세팅

이번 챕터는 별도 코드 실행이 없다. AI 코딩 도구의 프롬프트와 응답을 비교하는 게 핵심이다.

| 도구 | 용도 | 비고 |
|------|------|------|
| AI 코딩 도구 (아무거나) | 프롬프트 비교 실습 | Claude Code, Cursor, Copilot, ChatGPT 중 아무거나 |

직접 따라해보고 싶은 사람은 본인이 쓰는 AI 코딩 도구로 같은 프롬프트를 쳐보면 된다. 결과가 100% 같지는 않겠지만, 방향성은 똑같을 거다.

(AI 도구가 없어도 괜찮다. 이 챕터에서 보여주는 프롬프트/응답 비교만 읽어도 충분하다.)
